# Snuggle 서버 실행 가이드
**작성일**: 2024-12-14

## 1. 사전 준비
*   Node.js가 설치되어 있어야 합니다.

## 2. (추천) 한 번에 실행하기
가장 간편한 방법입니다.

1.  폴더 내에 있는 **`start_all.bat`** 파일을 더블 클릭하세요.
2.  검은색 창이 **하나만** 뜨고, 잠시 후 **크롬 창이 자동으로 열리며** 사이트에 접속됩니다.
    *   (창 내부에서 백엔드와 프론트엔드 로그가 같이 출력됩니다)

### Mac 사용자 (macOS)
1.  터미널을 열고 해당 폴더로 이동합니다.
2.  `sh start_all.sh` 명령어를 입력하세요.
    *   (처음 한 번은 `chmod +x start_all.sh` 명령어로 실행 권한을 줘야 할 수도 있습니다)
3.  자동으로 서버가 켜지고 브라우저가 열립니다.

---

## 3. 따로 실행하기 (수동)
문제가 생겼거나 따로 확인하고 싶을 때 사용하세요.

### 1단계: 백엔드 (Back-end)
1.  터미널에서 `back-end` 폴더로 이동: `cd C:\Snuggle\back-end`
2.  실행: `npm run dev`
3.  확인: `Server running on http://localhost:4000`

### 2단계: 프론트엔드 (Front-end)
1.  **새** 터미널에서 `front-end` 폴더로 이동: `cd C:\Snuggle\front-end`
2.  실행: `npm run dev`
3.  확인: `Ready in ...` 및 `http://localhost:3000`

## 4. 웹사이트 접속
*   **주소**: [http://localhost:3000](http://localhost:3000)

## 5. 서버 끄는 법
*   **Windows**: 켜져 있는 **검은색 터미널 창을 닫으면(`X` 버튼)** 실행 중인 모든 서버가 함께 종료됩니다.
*   **Mac**: 터미널에서 `Ctrl` + `C`를 누르면 서버가 종료됩니다.
